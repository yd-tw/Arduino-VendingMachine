## 技術細節

- 伺服馬達

## 製作歷程
- beta-1.0 軟體功能測試通過

## 使用方法
1. clone這個儲存庫
2. 於Arduino函式庫管理員安裝「Adafruit_SSD1306」並選擇安裝全部依賴項
3. 依照配線圖(尚未提供)連接電路
3. 編譯程式並上傳至開發板
